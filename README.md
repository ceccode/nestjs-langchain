# NestJS + Langchain



## Installation

```bash
$ npm install
```

## Running the app

```bash
# set node versione
$ nvm use

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
